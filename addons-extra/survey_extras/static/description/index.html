Overview

This app adds a template for Customer Satisfaction Survey (CSAT) and enhancements to Survey like conditional questions, regular expression based input pattern and file attachment question type. It also adds improvement to Survey web page layout.

Key Features


Thanking message of the survey and description of the page and question made visible

Add question type for File Attachment

Conditional questions based on the response to a particular multiple choice single answer question

Define regular expression based input pattern

Survey web page layout improvements

Customer Satisfaction Survey (CSAT)

Pre-loaded Customer Satisfaction Survey (CSAT) template with sample questionnaire

File Attachment & Conditional Questions

Set a condition to display a question based on the response to another multiple choice single answer question. File attachment question type is added which is one of the key features of certain surveys.

Regular Expression Input Pattern

Define a pattern to input values in the text box type questions. Ex: Mobile Number, Email, ID Proof Number etc.
